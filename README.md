# Pokemon Gallery

A React Pokedex app for the first 15 Gen 1 Pokemon. Browse individual stats with the carousel viewer, search by name, or explore the full gallery grid.

---

## Features

- **PokeScroll** — browse one Pokemon at a time with smooth fade transitions
- **Search bar** — filter by name with a live dropdown showing images
- **Gallery** — responsive card grid of all 15 Pokemon
- **Dark theme** — consistent dark UI with red accent color throughout
- **Page transitions** — smooth fade when switching between Home and Gallery

---

## Tech Stack

| Layer | Technology |
|---|---|
| Frontend | React 19, React Router 7 |
| Backend | Express 4, Mongoose 7 |
| Database | MongoDB |
| Styling | Plain CSS |

---

## Quick Start

```bash
npm install
npm start
```

Opens at [http://localhost:3000](http://localhost:3000). No backend or database needed to run the frontend.

For full setup instructions (environment variables, backend server, adding more Pokemon) see [GUIDE.md](GUIDE.md).
